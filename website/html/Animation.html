Panel Animation

<DuxPanel> supports CSS animation. All modern evergreen browsers (such as Firefox and Chrome) support CSS animation as well as Internet Explorer 10 or later. Simple entry and exit animations are built-in as well as support for keyframes and the CSS shorthand animation syntax.

Entry and Exit Animations

<DuxPanel> offers optional entry and exit animations. The following properties control animations when a panel is opening and closing. slideInFrom and slideOutFrom forces fadeIn and fadeOut to true.

Property Default Description
fadeIn false (true in DuxDialog) True if the panel and the backdrop fade in
fadeOut false (true in DuxDialog) True if the panel and the backdrop fade out
slideInFrom none ("top" in DuxDialog) "top", "left", "bottom", or "right"
slideOutTo none ("top" in DuxDialog) "top", "left", "bottom", or "right"

CSS Animation

For more complete control over animation, CSS animations are supported. The animation property contains an animation shorthand referencing @keyframes specified in the CSS.

If your animation modifies the CSS top or left properties, users will not be able to drag the panel as long as the animation property to DuxPanel is non-empty.
