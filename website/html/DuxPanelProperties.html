DuxPanel Property Reference

Property Default Value Description
allowClose true If true, a close button is placed on the right side of the title bar. Note, this property is ignored if title is empty.
allowDrag true If true, the user is permitted to drag the panel by the title bar.
animation (none) This is a CSS animation shorthand format, such as "cool-animation .25s ease".
center true If true, the panel is centered horizontally and vertically in the browser's viewport. Note, the viewport is the visible portion of the document body. If the viewport is resized, the panel is moved to keep it centered.
clickToDismiss true If true, the onClose function is called if the user clicks outside the panel. This property is ignored if modal is false.
fadeIn true If true, the panel's opacity is faded from 0 to 1 using a CSS animation when the show property is changed from false to true.
fadeOut true If true, the panel's opacity is faded from 1 to 0 using a CSS animation when the show property is changed from true to false.
height (none) The panel height. This property can be a number (in pixels) or a percentage of the browser viewport width such as "50%" or an object of viewport breakpoints. See the topic on responsive panels for more information. If this property is missing, the panel will be sized tall enough to accommodate its contents.
left (none) The left edge of the panel. This property can be a number (in pixels) or a percentage of the browser viewport width such as "50%" or an object of viewport breakpoints. See the topic on responsive panels for more information. This parameter can be omitted if center is true.
modal true If true, a backdrop covers the entire viewport preventing the user from interacting with the part of the web page visible outside the panel.
onClose (none) This function is called when the user clicks the close button on the title bar.
onEnterPressed (none) This function is called when the user presses the Enter key, but only if the panel is the most recently opened. This property is ignored if modal is false.
onEscPressed (none) This function is called when the user presses the Escape key, but only if the panel is the most recently opened. This property is ignored if modal is false.
show false If true, the panel is rendered in the DOM and is visible. If false, the panel is removed from the DOM.
slideInFrom (none) Can be one of "top", "right", "bottom", or "left".
slideOutTo (none) Can be one of "top", "right", "bottom", or "left".
title (none) The title to show in the panel's title bar. If this property is missing or empty, no title bar is rendered.
top (none) The top edge of the panel. This property can be a number (in pixels) or a percentage of the browser viewport width such as "50%" or an object of viewport breakpoints. See the topic on responsive panels for more information. This parameter can be omitted if center is true.
width (required) The panel width. This property can be a number (in pixels) or a percentage of the browser viewport width such as "50%" or an object of viewport breakpoints. See the topic on responsive panels for more information.
